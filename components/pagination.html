Pagination

Breaks a large set of results across pages. Four types — a connected button group, spaced clear buttons, a page count (Page X of Y), and dots — each with optional prev/next label and icon. Click through them: a single active indicator glides between pages, dots stretch into a pill, and prev/next nudge on hover.

Anatomy try it

Button group — prev · numbered pages · next, connected by hairline dividers. The active page fill (--btn/primary/default) slides smoothly as you click.
