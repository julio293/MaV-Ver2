Chart

A transaction line chart — a bold title, a type dropdown, and a smooth spline over Sun–Thu with a gradient area fill, dashed value guides, haloed data points and value labels. Four types: Loss (Expenses · red), Earn (Earnings · green), Flat (a level line · brand) and None (empty). The line colour routes through the --Line-Chart/* tokens, so the brand variant re-themes blue → lime in dark.

Interactive try it

Tap the dropdown to cycle the type — Expenses → Earnings → Flat → None. The spline morphs point by point, the values count to their new figures, and the colour cross-fades; the flat line settles level. Hover a point to lift it.
