Header / App Bar

The mobile top navigation bar. An iOS status bar sits above a 56px row with a leading back control, a centered screen title, and a trailing slot that takes nothing, a text button, one icon, or multiple icons. Comes in a default (light) and inverted (dark) appearance.

Anatomy

Status bar (44px) → nav row (56px): before back icon · centered title · after action slot. Surface reads from --Header-app bar/bg-header.
