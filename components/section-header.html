Section Header

The titled row that opens a section of content. A title with an optional description on the left, and an optional trailing action — a “See all” text link or an icon button. Three sizes, left / center alignment, default / subtle / inverted appearance, and a skeleton loading state. Type is Plus Jakarta Sans throughout.

Anatomy

Title + description (--main-text/default · --main-text/description) with a trailing “See all” link (--main-text/brand, re-themes to lime in dark). Gap 16px, title→description 4px.
