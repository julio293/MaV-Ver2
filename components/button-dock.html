Button Dock

A sticky bottom footer that keeps the primary action fixed above the home indicator. Rounded-top surface (32px), 16px padding, and a soft 12px elevation. Optional checkbox row, secondary button, and home indicator.

Anatomy

Surface + full-width primary button + iOS home indicator. Rounded top corners, --button-dock/bg-dock.

Button dock — 375px · radius 32px top · padding 16 · shadow 0 0 12Uses --button-dock/bg-dock + --btn/primary tokens
