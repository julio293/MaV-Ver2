Accessibility
✓ Pass
Color Contrast
Digit text on input background: 7.1:1 (light), 8.3:1 (dark). Both exceed WCAG AA minimum of 4.5:1 for normal text.
✓ Pass
Keyboard Navigation
Tab focuses the group. Arrow keys move between boxes. Backspace deletes and moves left. Enter submits the form.
✓ Pass
Screen Reader
Each input has aria-label="OTP digit N". Role="group" with aria-label="One-time password" on the wrapper.
✓ Pass
Paste Support
Pasting a 6-digit string fills all boxes at once. Works with SMS autofill on iOS and Android. Non-numeric characters are stripped.
✓ Pass
Focus Ring
3px focus ring with 3px offset. Visible in forced-colors/high-contrast mode. Not suppressed on :focus-visible.
⚠ Note
Error Announcement
Error messages must use role="alert" or aria-live="polite" to announce to screen readers. The component emits the DOM; parent must handle announcement.
